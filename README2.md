# Employee Attrition Analysis and Prediction Using Machine Learning

## Project Overview
This project analyzes employee attrition using machine learning. It includes data cleaning, exploratory data analysis (EDA), preprocessing, model training, evaluation, and prediction.

## Dataset
IBM HR Analytics Employee Attrition Dataset

## Tools & Technologies
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Models
- Logistic Regression
- Random Forest Classifier

## Evaluation Metrics
- Accuracy
- Classification Report
- Confusion Matrix
- ROC-AUC Score

## Project Outcome
Random Forest achieved better performance than Logistic Regression in predicting employee attrition.
